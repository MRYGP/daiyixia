支持功能:
- [x] 商家上传假发商品图
- [x] 上传正面照
- [ ] 上传侧面照
- [x] 正面生成
- [ ] 侧面生成
- [ ] 商品图保真(颜色/长度/刘海/分缝/卷度/蓬松)
- [x] 失败提示

当前不支持:
- 稳定侧脸、多角度和后脑勺试戴。
- 像照镜子一样的实时动态试戴。
- 商品图六维保真尚未达到稳定可验收标准。

当前最大问题:
- 静态试戴对输入照片要求严格，需正面、单人、脸部清晰、肩膀可见。
- 部分边界样本会出现人脸检测失败、肩膀不可见、角度过大等错误。
- 商品图保真、发际线和耳侧边缘融合仍需扩大样本测试。

下一步最小修复:
1. 使用更多真实商户假发商品图测试自助上传闭环。
2. 固定记录身份保持、发际线、边缘融合、商品保真和耗时。
3. 补充隐私说明和临时图片自动清理策略。
