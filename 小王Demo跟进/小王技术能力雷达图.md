# 小王技术能力雷达图

更新时间：2026-06-01

## 一、用途

本文件用于同步当前 Demo 已覆盖的技术能力、完成度和下一步优先级。评分范围为 1-5 分：

- 1 分：尚未验证。
- 2 分：完成初步调研。
- 3 分：最小链路可运行。
- 4 分：已具备可展示 Demo。
- 5 分：达到可商用稳定性。

## 三、当前技术路线判断

### 主线

```text
用户正面照 + 商户假发商品图 -> 静态 AI 试戴图
```

### 辅助方向

```text
已授权真人素材 + 假发商品图 -> 营销展示视频
```

### 暂不投入

```text
依赖通用生成模型反复调提示词
```

原因是通用生成模型容易重画整张人像，无法稳定保持用户本人身份。

## 四、阶段目标

下一阶段目标不是增加更多页面，而是提高验收稳定性：

1. 固定一组正面用户样本。
2. 收集一批真实商户假发商品图。
3. 对每张结果记录身份保持、发际线、边缘融合、商品一致性和耗时。
4. 汇总成功率，判断是否达到卖家现场演示标准。
