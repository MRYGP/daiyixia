# SK｜头发 AI 产品推进台 · Project 配置

## 一、Project 基本信息

**名称**:SK｜头发 AI 产品推进台

**描述**:
专门推进 AI 假发/头发试戴转化工具:管理 Demo、业内访谈、卖家验证、BP 迭代、漂移控制和 GitHub 入库。所有当前状态以 MRYGP/SK 当前文件为准。
