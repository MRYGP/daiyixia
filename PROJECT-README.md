# 戴一下·假发 AI 试戴 · Project README

## 一、当前主线

AI 假发 / 头发试戴转化工具。

**当前不是**:
- 脱发诊断
- 独立 C 端 APP
- 泛头发 AI
- 广告平台
- 多品类视觉验证 API

**当前主线是**:

> 嵌入假发卖货漏斗,用 AI 试戴解决"模特展示 ≠ 用户本人效果"的展示对象错位,
> 帮助卖家提升新客转化、降低退货。

---

## 二、当前阶段

**第一阶段：试戴图做像 + 嵌入卖家漏斗 + 验证降退货**。

**当前唯一问题**:

> 假发卖家看了 Demo 后,会不会认为它能帮自己提升转化、降低退货,并愿意继续试用或付费?
