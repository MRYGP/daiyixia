# 头发产品长远技术护城河与 5/22 后路径

## 一、头发产品当下的真实位置

**5/22 Demo 本身没有护城河**——任何人 3 天能复刻。
**所以这份文件回答的不是"Demo 怎么有护城河",
是"Demo 之后 SK 怎么慢慢长出护城河"**。

## 二、4 个台阶,SK 要爬

### 台阶 0(现在·5/22 Demo)
- 调通用 API + 包装界面
- 没有护城河,大厂 6 个月内置就死
- **但这是台阶 1 的入口,不能跳过**

### 台阶 1(6-12 个月后)
- 数据沉淀通道开始运转
- 用户照片 + 假发选择 + 真实反馈数据开始积累
- 用积累的数据做 LoRA 微调开源模型
- **护城河强度**:别人能抄技术,抄不到数据

### 台阶 2(12-24 个月后)
- 和许昌假发厂家签数据合作(他们给工厂级假发素材库)
- 和跨境头部卖家签合作(他们给真实退货数据)
- **护城河强度**:关系是排他的,新进入者绕不开

### 台阶 3(24-36 个月后)
- 不只"贴假发看效果",做"懂假发的诊断+匹配"
- 用户特征(脸型/头型/肤色)→ 适合假发的匹配规则数据库
- **护城河强度**:真正的诊断引擎,不再是图生图壳

## 三、为了爬上台阶,5/22 Demo 必须预埋的 3 件事

### 预埋 1·数据沉淀通道

5/22 Demo 不是"用户拍照→生成→关闭"就完了:

- 用户**授权后**,生成结果保留(脱敏)
- 用户**反馈**(满意/不满意/为什么)被记录
- 用户**最终是否下单**被追踪(通过卖家反馈)

**5/22 Demo 现在不做,以后补就难——隐私架构和数据架构是从第一天开始决定的**。

### 预埋 2·卖家合作预埋数据条款

> ⚠️ 5/19哨兵警示:"6个月免费换数据"与路径B"3-6个月现金流自给"存在直接张力。
> 5/22后必须先验证卖家是否愿意【直接付费】——免费换数据是B计划,不是A计划。
> 不要因为想要数据飞轮,就默认走免费路线。先问能不能收钱。

5/22 后杨找卖家试用时,**不是直接卖工具,是谈数据合作**:

> "我免费给你工具用 6 个月,你给我回流'用户试戴后是否下单 + 退货原因'数据。
> 这些数据归 SK,用于产品迭代。"

**6 个月免费换数据,这是 SK 启动期最重要的"价值交换"**。
对卖家:免费工具 + 不需要付费
对 SK:启动数据飞轮

### 预埋 3·API 用着但准备迁移

- 通义万相 / 国内 API → 黑盒,微调不了
- 开源模型(HairFastGAN / SD)→ 可以自己微调,长期能建数据飞轮

**5/22 短期用 API 没问题(快速验证),
中期(3-6 个月)如果数据够,切到开源模型自托管**。

哨兵评估文档(同目录"技术路线-哨兵评估参考.md")指出:
切到开源模型需要 1-2 周 + 可能外包。**这件事 5/22 后基于业内反馈再决定**。

## 四、5/22 给业内人士的真实回答

如果业内人士问"你们护城河是什么":

> "现在没有,Demo 谁都能做。
> 但我们走的不是'AI 工具'路径,是'懂假发的视觉决策能力'路径。
> 我用 6 个月给你免费,换你给我数据回流——这是我们的飞轮。
> 3 年后,如果你们行业的卖家想做 AI 试戴绕不开 SK,
> 不是因为我们技术更强,是因为我们的数据更准。"

**这是路径 B 求生存路径里"小公司怎么长出护城河"的真实答案**——
不是颠覆叙事,是慢慢长。

## 五、3 个反向警告(防漂移)

### 警告 1·台阶 0 必须先跑通,不能跳

不要听到"数据飞轮"就以为可以跳过 Demo。
**没有 Demo → 没有用户 → 没有数据 → 没有飞轮**。
台阶 0 是地基。

### 警告 2·"数据合作"是 5/22 后才谈的事

**5/22 前不要主动谈数据**——卖家会本能戒备。
先让卖家用 Demo,他认可效果后,再谈"你愿意给我反馈数据吗"。

### 警告 3·台阶 2/3 是涌现物,不是预设目标

不要写到 BP 里说"SK 3 年后会成为假发行业基础设施"——
这是 VC 故事写法,杨已经选了路径 B 求生存。

**对外说**:"我们做头发 AI 工具,3 年内现金流自给"
**对内做**:数据沉淀通道 + 合作数据条款 + API 迁移准备
**5 年后回头看**:如果做成了,就是台阶 2 / 3 的位置
