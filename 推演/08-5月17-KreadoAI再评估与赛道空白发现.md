# 头发产品推演(08)· 5/17 KreadoAI 再评估与赛道空白发现

## 一、研究动机

5/15-5/16 推演中,Claude 曾将 KreadoAI 视为"假发 AI 试戴"方向的潜在占位者或降维威胁。5/17 经深度研究(官网产品线、母公司背景、Similarweb / Trustpilot / G2 真实使用量级)后,**推翻该判断**。

## 二、KreadoAI 真实定位

**不是**认知断层修复产品,**不是**假发卖家站内转化工具。

**是**易点天下(EasyClick / 易点天下,上市广告营销集团)旗下的 **AIGC 广告素材工厂**——为出海电商 / 品牌方批量生产多语言营销素材:

| 维度 | 真实定位 |
|---|---|
| 母公司战略 | 易点天下 = 程序化广告投放 + 素材产能;KreadoAI 是素材侧的 AI 产能工具 |
| 核心用户 | 出海品牌运营、广告投手、素材团队 |
| 核心产出 | 数字人口播视频、AI 模特图、多语言广告片、商品讲解视频 |
| 商业模式 | SaaS 订阅 + 按量计费(素材生成),不是按商家转化率分成 |
| 与 SK 库铁律 19 的关系 | **寄生在广告素材生产漏斗**,不是寄生在假发购买漏斗 |

**结论**:用 SK 库语言说,KreadoAI 解的是"广告主做素材的效率问题",不是"假发准用户为什么没下单"的认知断层问题。两者用户、漏斗、付费方完全不同。

## 三、"虚拟试穿"在其产品线中的真实位置

KreadoAI 官网展示 **12+ 独立产品线/功能模块**(截至 2026-05-17):

1. AI 数字人口播视频(700+ 数字人形象)
2. 真人形象克隆(Avatar Clone,约 5 分钟素材)
3. 声音克隆(Voice Clone,多语言输出)
4. AI 模特生成(AI Model)
5. 商品讲解视频(Product Explainer)
6. AI 文案生成(AI Copywriting)
7. 文字配音 / TTS(40,000+ 音色,140+ 语言)
8. PPT / 图片会说话(Talking Photo / PPT Broadcast)
9. AI 直播(AI Live Streaming)
10. 智能导购(Intelligent Shopping Guide)
11. 智能抠图(Smart Background Removal)
12. 营销广告创意批量生成
13. **虚拟试穿 / Virtual Try-On**(假发、眼镜、美妆、服饰、珠宝等)
14. 多语言视频一键本地化

**"虚拟试穿"排在上述列表的中后段**,官网有独立落地页,但:
- 落地页叙事是 **"为电商生成高转化试穿广告视频"**,不是"嵌入卖家店铺提升站内转化"
- 同页并列眼镜 / 美妆 / 服饰 / 珠宝,假发只是 SKU 类目之一
- 产出形态是 **试穿效果视频素材**,供广告投放,不是 C 端买家在店铺里交互试戴

**判断**:虚拟试穿对 KreadoAI 是 **次要 feature / 垂直场景素材能力**,不是主线产品,更不是"为假发卖家做站内 AI 试戴 SaaS"。

## 四、真实使用量级(2026-05-17 采集)

| 指标 | 数据 | 解读 |
|---|---|---|
| Similarweb 月访问量 | ~190K/月 | 有流量,但是全球 AIGC 工具量级,非赛道霸主 |
| Trustpilot 评论 | 56 条 | 用户声量有限 |
| G2 评论 | 5 条 | 几乎无企业采购侧口碑沉淀 |
| 母公司体量 | 易点天下上市,市值可观 | **母公司大 ≠ 该产品线占住细分赛道** |

**与"占住赛道"的对照标准**:
- 占住 = 该细分方向的默认选择,竞品调研绕不开,商家访谈会主动提起
- KreadoAI 190K/月 + 个位数 G2 评论 → **有产品、有曝光,但未达"占住假发 AI 试戴 SaaS"量级**
- 其流量主要来自"AI 视频 / 数字人"通用词,不是"假发试戴"垂直词

## 五、翻转结论:赛道可能真的是空的

### 5.1 被推翻的旧判断

| 旧判断(5/15-5/16) | 新判断(5/17) |
|---|---|
| KreadoAI 可能已占位假发 AI 试戴 | KreadoAI 做的是广告素材,虚拟试穿只是副产品 |
| 需要抢窗口期避免被降维 | 降维威胁仍在(抖音/淘宝),但 KreadoAI 不是那个降维者 |
| 竞品地图里缺直接竞品 | **"为假发卖家做站内 AI 试戴 SaaS"很可能没有真正占位者** |

### 5.2 修正后的竞品空白

结合 01 竞品地图(四层)与本次研究:

**仍不存在的**(截至 5/17):
- 寄生在假发购买漏斗、以"展示对象错位"为第一卡点
- 付费方为卖家(转化率 / 退货率),不是广告投手
- 产出为买家在决策瞬间的交互试戴,不是广告素材视频

**KreadoAI 不占这个空**——它甚至不在同一漏斗上。

### 5.3 同步警示:空 ≠ 值得做

赛道空白是 **机会信号**,不是 **成功保证**。

观察:KreadoAI 作为"调 API + 包装界面"的 AIGC 工具,其真实天花板可能就是 **~190K 月访问 + 低评论沉淀** 这一量级——

- 技术门槛:虚拟试穿 API 已商品化(多家供应商)
- 差异化难:功能易被复制,用户忠诚度低
- 付费意愿:素材工具按量付费天花板清晰;转化工具需证明 ROI,销售周期长

**元层质疑(待杨补充)**:这类"简单 AI 应用"(API 聚合 + SaaS 界面)的赛道,是否整体上已经"满员"——即每个垂直都能做出产品,但都做不大?
