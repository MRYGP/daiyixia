## 1·问题 — 假发卖家在为什么流血?

### 1.1 真实痛点(报告 4,B 级证据)

中国假发卖家 TOP5 痛点(基于公开渠道频次统计):

| 排名 | 痛点                   | 出现频次 | 痛点强度 | AI 能否解 |
| ---- | ---------------------- | -------- | -------- | --------- |
| 1    | 新客获取贵且转化低     | 12       | 极高     | 部分      |
| 2    | 信任不足导致退货差评   | 11       | 极高     | **是**    |
| 3    | 多语种客服跨时区承接   | 10       | 高       | 是        |
| 4    | 内容制作与产品教育负担 | 9        | 高       | 是        |
| 5    | 质量稳定与供应链       | 8        | 高       | 部分      |

### 1.2 退货是真实的利润黑洞(报告 1,C 级证据)

行业退货数据(尚普咨询 1220 份样本):

- **行业平均退货率**:15-25%
- **优化后可控**:15% 以内
- **单新品最优**:8%
- **三大退货原因**:色差 42% + 头围不合 31% + 发质显假 18% = **91%**

**核心洞察**:这 91% 的退货原因,**全部与"购买前看不到自己戴上效果"相关**——这正是 AI 试戴能直接解的事。

### 1.3 行业利润损失估算(C 级推算)

- 中国假发市场 2024 年:国内 168 亿 + 出口 375 亿 = **543 亿**
- 按 20% 平均退货率算,**退货成本约 108 亿/年**
- 若 AI 试戴能把 1000 家中腰部卖家的退货率从 20% 降到 12%(8 个百分点),按每家年营收 500 万算:
  - 每家节省 = 500 万 × 8% = **40 万/年**
  - 1000 家节省总和 = **4 亿/年**
- SK 抓住这个空间的 0.5%-1% 即可达成路径 B 求生存目标

## 2·解决方案 — 第一代产品是什么

### 2.1 第一代 Demo(5/17-5/22)

**形态**:网页或扣子机器人(具体由技术决定) **核心能力**:用户拍照 → 选择假发款式 → 看戴上效果 **目标**:验证 AI 视觉能力能否做出"自然度过关"的假发试戴

**Demo 阶段不包含**:用户登录 / 付费 / 卖家后台 / 数据分析 / 美化 UI

### 2.2 第二代产品(5/22 后基于市场反馈定形态)

**当前 4 个候选形态(不预设,让市场说话)**:

| 形态             | 服务对象                    | 切入路径               |
| ---------------- | --------------------------- | ---------------------- |
| A · ToC 工具     | 假发购买决策中的消费者      | App / 小程序 / 网页    |
| B · B 端插件     | Shopify / 独立站 / 淘宝卖家 | SaaS 订阅 / 按调用收费 |
| C · 厂家增值服务 | 假发厂家给下游代理商/门店   | 白标 + 培训 + 工具     |
| D · 试戴即广告(根系想象) | 任何想做试戴的商家(假发/植发/接发/理发/美瞳/眼镜) | 试戴广告 SDK 嵌入平台广告位,CPT 或转化分成 |

> **形态 D 是 5/18 早涌现的根系想象,不是当前主线**。详见 `12-5月18-Demo成功后的两个根系.md`。第一代 Demo 跑通 + 第二代形态(A/B/C 选一)站稳后,可能演化出 D。5/22 业内人士访谈时,会用附加问题"如果这个试戴效果作为广告投放,你会感兴趣吗?"做交叉验证。

**5/22 后的决策依据**:

- Demo 的真实使用反馈
- 8 个真实业内人士的反应
- 哪种形态卖家说"我愿意付钱"

## 5·商业模式 — 谁付钱,付多少,为什么付

### 5.1 付费方支付意愿(报告 4,A 级价格锚点)

假发卖家已经在为以下工具付费:

| 类别        | 工具样本              | 定价       |
| ----------- | --------------------- | ---------- |
| 建站 SaaS   | Shopify 基础版        | $39/月     |
| 建站 SaaS   | 店匠                  | $28-218/月 |
| 客服外包    | Callnovo 共享英语客服 | $650/月    |
| AI 内容工具 | KreadoAI 月订阅       | $6-25/月   |
| AI 模特图   | KreadoAI 按张         | $3-4/张    |

**含义**:卖家不是不愿付费,是**只愿为"直接提高转化 / 降低退货 / 替代人工"的东西付费**——SK 切的正是这一点。

### 5.2 SK 第二代定价方案(待验证基线)

| 套餐 | 月费(RMB) | 包含                                            |
| ---- | --------- | ----------------------------------------------- |
| 基础 | 99        | 每月 100 次试戴 / 5 款假发素材 / 基础品质       |
| 标准 | 299       | 每月 500 次试戴 / 30 款素材 / 高品质 / 邮箱客服 |
| 高级 | 699       | 每月 2000 次试戴 / 无限素材 / 优先客服 / API    |
| 企业 | 议价      | 白标 / 厂家给代理商场景 / 私有部署              |

**定价依据**:略低于 KreadoAI($19-499/次)但高于扣子(免费),覆盖中腰部卖家可承受区间。

### 5.3 收入模型(路径 B 求生存)

**关键数字**:

- 月营收临界点:**20 万 RMB**(= 现金流自给的最低门槛)
- 按平均 200 元/月/客户计算:**需 1000 付费卖家**
- 中国跨境假发卖家(许昌 3000 家备案)+ 国内电商假发卖家(估算 5000-8000 家)= **可触达池约 1 万家**
- SK 需获客 1000 家 = **10% 渗透率**——这是路径 B 现实数字,不是 1% 渗透率(VC 故事)

**3 年路径**:

- 12 个月:500 付费卖家,月营收 10 万
- 18 个月:1000 付费卖家,月营收 20 万(**现金流自给临界点**)
- 36 个月:3000 付费卖家,月营收 60 万,年营收 720 万

## 9·哲学:为什么是路径 B,不是 VC 故事

### 9.1 SK 不是创业明星,是 2-5 人小团队

- 不画 10 亿美元市场饼
- 不追求 10 倍回报
- 目标是 3 年内现金流自给,然后慢慢长
- 这是"小公司能生存"路径,不是"亚马逊从卖书开始"路径

### 9.2 接受涌现,不预设

- 第一代 Demo 不预设服务谁(老客 / 新客都行)
- 第二代形态不预设(ToC / B 端 / 厂家增值由市场说话)
- 破坏式/颠覆式创新可能涌现,但不预设

### 9.3 给跟投人/资源伙伴的明确信号

- **不要 VC**(VC 要 10 倍回报,SK 给不了)
- **要产业投资人 / 跟投人**(愿意陪 3-5 年,愿意接受年化 20-50% 回报)
- **要资源伙伴**(假发产业人脉 / 跨境卖家资源 / 技术合伙人)
- **要合伙人**(愿意一起押注 5-10 年的同频者)
